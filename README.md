## 字体调整
```markdown
 _倾斜_   *倾斜*
__加粗__ **加粗**
~~删除~~ 
```

>__加粗__ **加粗**
>~~删除~~

## 有序列表
```markdown
1. 淀真嗣
2. 明日香
3. 凌波丽
```

>2. 明日香
>3. 凌波丽

## 无序列表
```markdown
- 淀真嗣
* 明日香
+ 凌波丽
```
>- 淀真嗣
>* 明日香
>+ 凌波丽

## 分割线
```markdown
*****（星号）
-----（减号）
_____（底线）
```
>*****
>-----
>_____
